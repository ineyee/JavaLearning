## 一、通用命令

* 查询数据库里数据的总条数

```
DBSIZE
```

* 查询数据库里所有的 key（生产环境不建议使用，因为会扫描所有的数据，阻塞其它命令的执行）

```
KEYS *
```

* 查询数据库里是否存在某个 key，存在返回 1、不存在返回 0

```
EXISTS $key
```

* 给某个 key 设置有效时间、单位秒，过期后会自动删除

```
EXPIRE $key $seconds
```

* 给某个 key 设置永久有效

```
PERSIST $key
```

* 查询某个 key 的有效时间，>= 0 代表有效秒数、-1 代表永久有效、-2 代表没有这个 key

```
TTL $key
```

* 查询某个 key 的数据类型

```
TYPE $key
```

* 删除某个 key，成功返回 1、失败返回 0

```
DEL $key
```
